## Stock Portfolio Tracker

## Description
The Stock Portfolio Tracker is a simple Python application that helps users calculate the total value of their stock investments. The user enters the stock name and quantity, and the program calculates the total investment based on predefined stock prices. The final result is also saved in a text file for future reference.

## Features
- Displays a list of available stocks.
- Accepts stock names and quantities from the user.
- Calculates the investment value for each stock.
- Calculates the total investment amount.
- Saves the final investment summary in a text file.
- Handles invalid stock names with an error message.

## Concepts Used
- Python Dictionary
- Loops (while, for)
- Conditional Statements (if-else)
- User Input
- File Handling
- Variables and Arithmetic Operations

## How to Run
1. Install Python 3.
2. Open the project in VS Code.
3. Open the terminal.
4. Run the following command:
   python task2.py
